Versatile tool that allows you to create and print flags directly in your console. With a user-friendly interface, you can build custom flags by choosing colors, patterns, and even adding a coat of arms.

## Key Features

1. **Stripe Customization:** Choose between horizontal and vertical stripes.
2. **Color Customization:** Select unlimited colors to create unique flag designs.
3. **Coat of Arms Integration:** Personalize your flag with a custom coat of arms image.
4. **User-Friendly Interface:** Enjoy a simple user interface for seamless flag creation.
5. **Versatile Flag Designs:** Experiment with different settings to create a wide range of flag designs.
6. **Examples for Inspiration:** Explore provided examples to spark creativity and serve as a starting point for your designs.
7. **Cross-Platform Compatibility:** Build and run your flags on any platform with ease.

## Code

1. **Function bufferedImageToStringArray:** Converts the coat of arms image into an array of strings. Each string in the array represents a row of pixels from the coat of arms image. It creates a strings that mark transperent and visible pixels.

2. **Function createFlag:** The flag is saved as an array of strings and can be printed using a for each loop or by calling a print() function from the Flag class. It creates empty characters for the flag stripes and full characters for the coat of arms.
